# Oxford 3000 Study

Oxford 3000のデータを使った、個人学習用の英単語カードアプリです。

## 機能

- 3,809語義レコードの検索とCEFR・品詞フィルター
- 英日・日英の単語カード
- 4択クイズ
- ブラウザ内の学習履歴保存
- 履歴のJSON書き出し
- スマートフォン・デスクトップ対応

## データと識別子

同じ単語でも品詞や語義が異なる場合は別レコードとして扱い、学習履歴は単語文字列ではなく`id`で紐付けています。

## 公開上の注意

このリポジトリは個人学習目的の非公式プロジェクトです。Oxford University PressおよびOxford Learner's Dictionariesとは提携していません。各レコードの参照元はアプリ内の`source_url`から確認できます。
